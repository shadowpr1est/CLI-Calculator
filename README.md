# Terminal Calculator in Go

A simple terminal-based calculator written in Go that supports:

- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`*`)
- Division (`/`)
- Square root (`sqrt`)
- Exponentiation (`^`)
